Vector Graphics 1

This assignment was my first time working with vector graphics using Adobe Illustrator. In this project, I practiced basic design skills such as changing the size of shapes, adjusting stroke weight, and using different colours to create a complete design.

This was my favourite assignment in the unit because it was the first project I completed, and I spent a lot of time experimenting with different shapes and layouts. Even though it may not be the most visually advanced piece, it was an important learning experience and helped me build a strong foundation for future vector graphics projects.

Vector Graphics 3 (Pen Tool Practice)

This assignment focused on practicing the pen tool in Adobe Illustrator. Overall, the two designs were not very difficult. The heart shape was easier to create because it mainly used smooth and simple curves.

The dog design was more challenging, especially at the turning points and when adjusting the layer order near the end of the project. This assignment helped me improve my control of curves and better understand how layering works in Illustrator.

Vector Graphics Summative Assignment (Ye Zhou Portrait)

In this assignment, I created a vector portrait of the celebrity Ye Zhou using Adobe Illustrator. I began by setting up the canvas, importing a reference image, and adding proportion guidelines to help with the facial structure.

I used the pen tool to block out the main shapes, including the face, hair, clothing, and background, and applied flat colours without outlines. I then added shadows and highlights using the Pathfinder tool to build form. One challenge was placing shadows naturally, which I overcame by reviewing classroom tutorials. Overall, this project helped me improve my Illustrator skills and better understand shading and portrait design.
